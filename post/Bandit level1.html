Bandit Write-up Level 1

Posted by CLoveYC on April 2, 2024

Goal:

Using command to get the password which is stored in a file called 'readme'

My Solution:

According to Bandit Command 1 to 6, simply use 'ls' to see what we have. When I type 'ls', the system outputs a file called 'readme', which is what we are looking for. So, I use the 'cat' command to see what's inside. After typing 'cat readme', we get a message. And that's the end of this level. We got the key: NH2SXQwcBdpmTEzi3bvBHMM9H66vVXjL.
